【试题解析】

试题分析：考查情景交际。A项意思是：当然，由你决定；B项意思是：当然，没问题；C项意思是：对不起，（一点钟）不行；D项意思是：对不起，我今天不行。问句是表示征求意见，而根据应答句的后一分句Will two o’clock be OK可知一点钟不行，故选C。句意：--今天下午一点我们能出去喝一杯吗？--对不起，一点不行。两点可以吗？
